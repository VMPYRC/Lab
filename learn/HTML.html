Images

Image Styles

  • These styles apply to other HTML elements too.
  • Images are used as an example to showcase the differences, because it is easier to demonstrate.
  • The priority is: Inline >> Internal >> External

Inline - Within the img tag on separate lines

      <img
      src="image.png"
      alt="Alternative Text Here"
      width="100"
      height="100"
      />

Inline - Within the style attribute of the img tag

      <img
      src="image.png"
      alt="Alternative Text Here"
      style="width: 100px; height: 100px"
      />

Internal - Within the head section of the HTML page

      <head>
        <style>
          img {
            width: 100%;
          }
        </style>
      </head>

External - Within the styles.css file

      img {
      border: 1px solid #000;
      width: 100px;
      height: 100px;
      background-color: #FFFFFF;
      width: auto;
      margin: 20px auto; 
      }
